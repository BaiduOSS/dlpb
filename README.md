# 项目名称
简要说明:  这是面向深度学习初学者的Paddle训练营中所用的参考源码

## 快速开始
git clone代码后，即可进入各个子目录并且运行。

注意：
 * _with_paddle的程序需要在PaddlePaddle的环境中运行
 * _with_numpy的程序需要在安装了numpy的python环境下运行
